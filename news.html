Overview A cool jQuery plugin for creating own news slider easily.

Own content or RSS feed

You can create the content by writing some HTML in your document or set a RSS feed and the content will be generated from the feed.

Responsive and Touch-Enabled

The slider is responsive and also supports mobile gestures to scroll the content in the full post view.

Refer from a preview to a website

You can enable a link on every preview, so when the user click on a preview, he will be redirected to a website.

Vertical or horizontal scroll

You can choice between 2 different scrolling directions - horizontal or vertical. You can also enable the infinite loop and hide the arrows.

Videos and Images support

The full post view also supports images and videos, so you can also use youtube embeddings. HTML tags are also supported.

Options to customize the slider

The plugin comes with a handful options, that allows you to customize the slider by your needs. The CSS is also well documented.

Examples See some sliders in action and how the plugin can be used.

Slider with RSS feeds

In this slider the content is created from a RSS feed.
You can also test your own RSS feed by adding your RSS feed URL:


Javascript Code:
$('#fancy-news-rss').fancyNews({width: 700, height:510, previewsPerPage:5, maxWords:45, feed:'http://feeds.feedburner.com/nettuts?format=xml'});


Slider with feeds and links

In this slider the content is created from a RSS feed and the links on the preview elements are also enabled. A slide time is set to 3000ms (3 seconds).

Javascript Code:
$('#fancy-news-links').fancyNews({width: 500, height:300, maxWords:25, feed:'http://feeds.feedburner.com/nettuts?format=xml', infiniteLoop: true, slideTime: 3000, useLinks: true, arrows: false});

Options The plugin comes with a lot of options for customizing it by your needs.

Available Options

Option Explanation Default Value
width The width for the slider in px. 500
height The height for the slider in px. 420
offset The offset between the previews in px 2
maxWords The number of words for the preview text. 55
previewsPerPage The number of previews per page. 3
numOfEntries The number of blog entries to load (only for RSS feed). 10
slideTime The slide time in ms. 0
feed An URL to a RSS feed. ''
backgroundColor The background color for all elements. '#1A1A1A'
backgroundOverColor The background color when you move your mouse over a preview element. '#0C0C0C'
useLinks Set it to true to enable the links on preview elements. false
targetWindow The target for the links, choice between '_self' or '_blank'. '_blank'
Since V2.0:
infiniteLoop		 Set it to true to enable the infinite loop when scrolling. When enabling this option, it will also hide the post count element. false
Since V2.0:
arrows		 Show(true) or hide(false) the navigation arrows. true
Since V2.0:
vertical		 Enable(true) the vertical scrolling. false
Since V2.0:
onListCreated		 A callback function that will be called when the preview list created. function() {}
